# URL Redirects

Use these redirects in Cloudflare Bulk Redirects to map old help center URLs to the new docs site.

## Getting Started

| Source URL | Destination URL |
|------------|-----------------|
| `/help-center/getting-started/how-do-i-create-my-custom-trained-chatgpt-chatbot` | `/docs/getting-started/create-your-chatbot` |
