You want to embed your custom trained AI support agent in your **Skywork.ai** website page. Here's how to add an AI support agent to Skywork.ai with SiteSpeakAI.

**Skywork.ai** is an AI-powered website builder designed to help users create websites quickly and easily. It leverages artificial intelligence to simplify the design and development process.

You can learn more at: [https://skywork.ai/](https://skywork.ai/)

## Add your AI support agent to your Skywork.ai website

### In SiteSpeakAI

Log into your SiteSpeakAI account and select your agent from the dropdown menu at the top, click on the **Install Agent** section in the sidebar, select the **Javascript** tab and then click on the **Copy Code** button to copy your agent code snippet.

![Copy code snippet](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/G0xBJY1kStrMxFTlBm4nKnm7AP6UqiEcdv1OUxTX.png)

### In Skywork.ai

1. Log into your **Skywork** account and select your project in the lefthand sidebar under **Projects**

![Skywork dashboard](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/9Vj7NABJJoX7SQ6mfKDdFSV00i0ivkCnyvk3Wl8O.png)

2. When the project opens, click on **Edit**

![Edit project](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/pqXl70N7YoLCbIL6WenopekcQMeqZS47YGeKwXXY.png)

3. Select the enlarge icon to enlarge the AI command line area

![Enlarge command line](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/zB8CLuhpChh4sbixTBw8ebGVeV6UrjOsYYfcVPdX.png)

4. Enter the following instruction/request into the AI command line:

**Please modify my code to include my SitespeakAI code snippet:** "*--- insert the AI support agent js code copied at the beginnng of the tutorial ---*" and then click on the **Submit** button. **Skywork AI** will start adding the code to your website, it takes a while so please be patient.

![Submit request](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/sMBRgaBKG42ZdHPsdBzIwOkM9Z3G7E2MRSZZkMoh.png)

5. When the process is completed, the AI support agent button will be displayed on you project page. Click on **Save** to save your changes to your live website.

![Save changes](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/eJyGO28QmMFhIJIqpmSGX4j5fd2KFgPfkUymR3bY.png)

6. Your AI support agent button will now be displayed on your **Skywork.ai** website page

![Agent button displayed](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/FBbCzqjOYbRe6v4PIGC0jfk5afbqKXtULD91cm6p.png)

7. If you click on the button it will display the AI support agent dialogue

![Agent dialogue](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/iOsm39J3Od3LhG4BRcT28Nlo39KhJx30rvYBbper.png)

Your AI support agent has now been added to your **Skywork.ai** website and your visitors/customers can start interacting with it!
