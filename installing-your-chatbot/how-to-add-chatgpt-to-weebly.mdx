You want to embed your custom trained chatbot in your Weebly website. Here's how to add ChatGPT to Weebly with SiteSpeakAI.

## Add your chatbot to your Weebly website

<InstallCopyCode />

### In Weebly

On your dashboard select **Edit site**

![Weebly - Select Edit site on your dashboard.PNG](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/EBqAqFERLkuJOVmRakseFL5mGmJmKHj6A1h5vTUr.png)

1. Drag the **Embed Code** element to your website page. **Note!** This must be done for each page you want to add your chatbot to.

![Weebly - Drag Embed Code to website page.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/y2aEerbFWzmwzWDydbtbFtPuezXoDVJhfqv7BSJg.png)

![Weebly - Embed Code element on website page.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/VogqnUXaIPZBYUn0MFyduaqRt8PnPX8yiSoqOLUN.png)

2. Click on the **Embed Code** element to open the **Custom HTML** editor.

![Weebly - Embed Code modal on website page.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/GFv1OAoldIOqv3XUfBkRpitqc3QaUG6wj0FIIqLd.png)

3. Paste your snippet in the editor text box and click on **Publish** to view your live website page.

![Weebly - Snippet Code in Embed Code modal on website page.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/jrEVsfWQgkBpM6sOsR065Opkxx3SS1Y6iKa8GOPK.png)

4. Your chatbot button will now be displayed on your website page.

![Weebly - Chatbot button on website page.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/QsojnRZu6PBax7eoCg2Z5246KiCfAWYnj7Iq3DU8.png)

5. If you click on the button it will display the chatbot dialogue.

![Weebly - Chatbot dialogue on website page.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/8ZPaB8iBdmOQSlSkU3jj17IpgXQ1hMuyNTGrdCHT.png)

Your chatbot is now added to your Weebly website and your customers can start interacting with it.
