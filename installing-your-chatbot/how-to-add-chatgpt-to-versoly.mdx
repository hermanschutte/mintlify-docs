You want to embed your custom trained AI support agent in your **Versoly** website page. Here's how to add an AI support agent to Versoly with SiteSpeakAI.

**Versoly** is a website builder that focuses on making it incredibly easy to create beautiful, professional-looking websites without needing any coding knowledge. It's built around a drag-and-drop interface, offering pre-designed templates and a library of customizable elements like images, fonts, and buttons. It's particularly known for its strong focus on **e-commerce**, making it a great choice for selling products online.

**Website:** [**Versoly**](https://versoly.com/)

## Add your chatbot to your **Versoly** website

### In SiteSpeakAI

Log into your SiteSpeakAI account and select your agent from the dropdown menu at the top, click on the **Install Agent** section in the sidebar, select the **Javascript** tab and then click on the **Copy Code** button to copy your agent code snippet.
![Copy code snippet](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/1hDAuh4DZqUI6TomPoruHgXlzDCE5Zht952LzElY.png)

### In Versoly

1. Log into your **Versoly** account and on the dashboard select the project you want to add your AI support agent to
![1 - Versoly - Select your project.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/3x7GZve713AK7iBPE5heSFVUcGjTamAY7rpIBFz5.png)

2. On the editor page select **Pages** on the top menu bar
![2 - Versoly - Select Pages on the menu bart.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/nHAobesk61q0QEuN85QJ0GMV0p8xHwDMEQrAdJu3.png)

3. Select the page you want to add your snippet to
![3 - Versoly - Select your page.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/41xjrbqpIaS4k4KwIfFCbZodBviZxizrCAEdBjBR.png)

4. Click on **Settings** on the top toolbar
![4 - Versoly - Click on Settings.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/bEhEyP1sKWHTO0B89wkmJNKLiuI21Hf2F1guu0JW.png)

Note that **Versoly** allows adding your code snippet **per page** as well as **globally to all pages**.

**A. Using the per Page option**:

1. Select **Page Settings**
![5 - Versoly - Select Page Settings.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/0HkIiJeti1fD8sqrTEfuaAThYuUFrKvMN8UYGuXj.png)

2. Select the **Advanced** tab, insert your snippet in either the **External Scripts** or the **Head Content** input text box and click on **Save Page**
![6a - Versoly - Insert snippet - External Scripts (Page).png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/7L2iSG7FxowjQY9388qMqRNGMOgn5aRK3rHZfErI.png)

![6b - Versoly - Insert snippet - Head Content (Page).png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/s3CGknGZMY1TESsbtbTX7QqKEZESCfFwxxcfup3X.png)

3. Click on **Publish**
![7 - Versoly - Click on Publish.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/K83W1ULl6F0PrcfDX5mZd6ZDKxE7ecWXrORqatYO.png)

**B. Using the global Site option**:

1. Click on **Site Settings**
![10 - Versoly - Site Settings.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/UAFfws58NmEzqLsq5MZyMKxInI82SELb55jpXMCJ.png)

2. Select the **Advanced** tab, insert your snippet in either the **External Scripts**, the **Head Content** or the **Site JS** input text box and click on the **Save** button for the respective option
![11a - Versoly - Insert snippet - External Scripts (Site).png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/tjituvJO48Bzyo5WDxD8X3cuEP7zsVB8p9rDZwW0.png)
![11b - Versoly - Insert snippet - Head Content (Site).png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/PFLasYeqirAPF18bAl9agUhuhcEvfstwkliipISE.png)
![11c - Versoly - Insert snippet - Site JS (Site).png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/T2fAhtKE3oEQNeiHJp6coKxOcL8qwLF4IuuTjsz3.png)

Please note that if you use the **Site JS** option, you must remove any **HTML** remark lines (`<!-- -->`). You must also remove the script tags, **Versoly** adds its own script tags when it adds your snippet to the site pages.

3. Click on **Publish**
![7 - Versoly - Click on Publish.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/q9KqklkX6IdtktqkGAZsj2kjw5fSywWGtmR5eqti.png)

**C. For both options**:

1. Your AI support agent button will now be displayed on your **Versoly** website page
![8 - Versoly - Chatbot button.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/8bODUw6OjODZkVIfwc1NtkAaKmlGnW0imal8uJ20.png)

2. If you click on the button it will display the AI support agent dialogue
![9 - Versoly - Chatbot dialogue.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/sLMRbu8jwXr8Moz1PKcLYXxoVKlycn60HjFQRg3D.png)

Your AI support agent has now been added to your **Versoly** website and your visitors/customers can start interacting with it!
