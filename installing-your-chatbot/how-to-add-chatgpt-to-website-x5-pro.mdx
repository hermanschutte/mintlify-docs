You want to embed your custom trained AI support agent in your **WebSite X5 Pro** website page. Here's how to add an AI support agent to WebSite X5 Pro with SiteSpeakAI.

**Website X5 Pro** is an easy and intuitive offline software that allows anyone to independently create small or medium-sized websites, blogs, and online stores.
More than 100 templates and lots of content will help you get started right away.

*   **AI-Powered Design:** Offers AI assistance for creating layouts and content.
*   **eCommerce Capabilities:**  Includes robust tools for selling products online.
*   **Advanced Customization:** Provides granular control over design elements.
*   **Landing Page Builder:**  Specialized tools for creating high-converting landing pages.
*   **No Coding Required:**  Designed for users with no technical skills.


**Website:** [**Website X5 Pro**](https://www.websitex5.com/)

## Add your AI support agent to your **WebSite X5 Pro** website

### In SiteSpeakAI

Log into your SiteSpeakAI account and select your agent from the dropdown menu at the top, click on the **Install Agent** section in the sidebar, select the **Javascript** tab and then click on the **Copy Code** button to copy your agent code snippet.
![uploads/help-center/images/8KXK6OLtFfNMredTkaNlDI45PfGlRwwondZFAVaW.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/8KXK6OLtFfNMredTkaNlDI45PfGlRwwondZFAVaW.png)

### In WebSite X5 Pro

*Update: You can now easily install SiteSpeak in WebSite X5 using a dedicated [marketplace plugin](https://guide.websitex5.com/en/support/solutions/articles/44002627853?_ga=2.39827555.1356160208.1760534545-779015879.1760534176).*

1. Log into your **WebSite X5 Pro** application and select the project you want to add your support agent to. Click on **Next** to continue
![uploads/help-center/images/Ee2NsRdY5CjP0Ktqj5nfkIapBKm0mhSPOqdVxQLX.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/Ee2NsRdY5CjP0Ktqj5nfkIapBKm0mhSPOqdVxQLX.png)

2. Select **Pages** on the left sidebar. **Please note** that if your website contains multiple pages you will have to add the support agent snippet to all pages. Adding an **HTML Code Object** in Website X5 Pro is per page and not website wide. Drag the **HTML Code Object** onto the page
![uploads/help-center/images/3mKdj6CYSwEacxdqPaZDediu02lDxEbw67y8nmbG.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/3mKdj6CYSwEacxdqPaZDediu02lDxEbw67y8nmbG.png)

3. Insert your snippet into the **HTML Code Object** and click on **OK** to continue
![uploads/help-center/images/cjq6MkYwkUBr1bmYvqyJ0Gc49AzFJJf8CR5xZ9uJ.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/cjq6MkYwkUBr1bmYvqyJ0Gc49AzFJJf8CR5xZ9uJ.png)

4. Click on **Save** to save the changes to your project
![uploads/help-center/images/jmxvOcgSSqjQfoudjCXkWsKzTuP8bEGx3gN58l28.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/jmxvOcgSSqjQfoudjCXkWsKzTuP8bEGx3gN58l28.png)

5. Click on **Preview** to preview your webpage
![uploads/help-center/images/DpkJwFjp4g7NTiSsOPj16w6InzEpjc7QIxHJYDEH.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/DpkJwFjp4g7NTiSsOPj16w6InzEpjc7QIxHJYDEH.png)

6. The AI support agent button displays correctly on the Preview page but clicking on it won't display the support agent dialogue. To see the dialogue box, select the **Preview the Website on another browser** option and select your browser of choice
![uploads/help-center/images/BNRD3d0dLnVvRro8h7lhyw0TbJJmqG67Nr9aKRel.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/BNRD3d0dLnVvRro8h7lhyw0TbJJmqG67Nr9aKRel.png)

7. If you click on the button it will display the AI support agent dialogue
![uploads/help-center/images/cdF6vvAHE9FvqwMMsz7AqFTY61aGXW95Og62CqQk.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/cdF6vvAHE9FvqwMMsz7AqFTY61aGXW95Og62CqQk.png)

Your AI support agent has now been added to your **WebSite X5 Pro** website and your visitors/customers can start interacting with it!
