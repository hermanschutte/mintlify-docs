You want to embed your custom trained AI support agent in your **GRAV** website page. Here's how to add an AI support agent to GRAV with SiteSpeakAI.

**GRAV CMS** is a modern open source flat-file CMS and it makes use of Markdown. Before we start adding our AI support agent, we have to do some setup to the configuration to allow your code snippet to be added.

**Please Note!** For GRAV, you have to add your AI support agent snippet to each page you want it to be displayed on, this means that the below instructions will have to be repeated for each page.

**Website URL:** [https://www.grav.io/](https://www.grav.io/)

## Add your AI support agent to your **GRAV** website

### In SiteSpeakAI

Log into your SiteSpeakAI account and select your agent from the dropdown menu at the top, click on the **Install Agent** section in the sidebar, select the **Javascript** tab and then click on the **Copy Code** button to copy your agent code snippet.
![uploads/help-center/images/PlJ1M7hDYjtgekYrC0WxIlvve2BqDkeQSpA2CRuV.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/PlJ1M7hDYjtgekYrC0WxIlvve2BqDkeQSpA2CRuV.png)

### In GRAV

1. Log into your **GRAV** account and on your dashboard, select **Configuration** on the left sidebar.

![1 - GRAV - Select Configuration.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/2qXhvDJ1FCVqVHOmpm9mHAvQRkEqDcl9t4Ek3HuV.png)

2. Select **Content** on the System configuration list and under **Process**, make sure **Twig** is checked and **Markdown** is unchecked.

![2 - GRAV - Select Content and make sure Twig is checked and Markdown not.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/vGmBAMZBkWXiphrvZwFmlsDqBnQ6t7N6iZV3oys9.png)

3. Click on **Save** (Top Right) to save your changes if any.

![3 - GRAV - Click on Save to save your changes if any.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/0aIcJjfhyA3WxwO5wf5hHeJvTDWZfT34tjNnL8np.png)

#### Let's insert our chatbot snippet

4. Select **Pages** on the left sidebar.

![4 - GRAV - Select Pages.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/dNky0G1FT86n3eoWxbXkHglNf8aNZ7Equ48Q1gFo.png)

5. Select the page you want to edit to add your chatbot snippet to.

![5 - GRAV - Select the page you want to edit.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/dfDLa54R3JkiutXIKDBqf3TIDziqOprlDJnl3YTb.png)

6. On the editor page, select **Normal** (the **Expert** option allows for more settings that we are not going to use now).

![6 - GRAV - Select Normal.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/KX2jBFkmHESN0kNSLj68osTipWpkWaFvR2Au58Rz.png)

7. Insert your snippet above (or below) any HTML content in the editor input dialogue box.

![7 - GRAV - Insert your snippet.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/tAmLFBfAMFzjcyuaxaS7x8t9KrQeqSOcwfu1LjNc.png)

8. Click on **Save** to save your input. **GRAV** will display a warning regarding potential XSS issues in content. This is due to the fact that the snippet contains a JS script tag. You can ignore this warning.

![8 - GRAV - Click on Save to save your input.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/RBGB41ChYAWLeBuRlQCmHsEes0AaU7wq8D8ORPMO.png)

9. Your AI support agent button will now be displayed on your **GRAV** website page

![9 - GRAV - Chatbot button.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/aMDzcIYyDaT6omQiM4XTGT1BUYiFjkrwpT6oMtKu.png)

10. If you click on the button it will display the AI support agent dialogue

![10 - GRAV - Chatbot dialogue.png](https://nyc3.digitaloceanspaces.com/cdn.sitespeak.ai/uploads/help-center/images/08ZEfr4bjxyPKLpQqbD7wbQskxKaEfqWo62m6LSB.png)

Your AI support agent has now been added to your **GRAV** website and your visitors can start interacting with it!
